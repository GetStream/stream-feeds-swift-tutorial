## Stream Feeds iOS tutorial

This is the final version of the tutorial. To follow along:

- Checkout and go to the initial commit:

```
git clone git@github.com:GetStream/stream-feeds-swift-tutorial.git
git checkout initial-commit
```

- Follow instructions from: https://getstream.io/activity-feeds/sdk/ios/

If you just want to run the final tutorial, provide your credentials in `UserCredentials.swift`.
